# About MessageBird

MessageBird is a third party service that provides APIs to send messages with Whatsapp.

> [!NOTE]
> MessageBird doesn't provide verify on-Whatsapp phone number but we can work around by trigger sendMessage API then parse the response to get the phone 's whatsApp account status.

## Requisites

* Create a MessageBird account [here](https://app.bird.com/auth/sign-up).

* Buy a phone number to send and receive message on Whatsapp via MessageBird.

## API

Reference API docs [here](https://docs.bird.com/api/channels-api/api-reference/messaging)

## Fee

Contact-Based Pricing + Message-Based Pricing [link](https://bird.com/pricing)

3000 contact - $45/month - up to 30,000 Email, SMS or WhatsApp sends
10000 contact - $100/month - up to 100,000 Email, SMS or WhatsApp sends

> [!NOTE]
> Doesn't charge on failed messages
