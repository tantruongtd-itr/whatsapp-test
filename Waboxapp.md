# About Waboxapp

Waboxapp is a third party service that provides APIs to send messages with Whatsapp.

> [!NOTE]
> Waboxapp doesn't provide verify on-Whatsapp phone number but we can work around by trigger sendMessage API then parse the error message to decide whether we send alert via message or not.

## Requisites

* Create a Waboxapp account [here](https://app.bird.com/auth/sign-up).

* Link your Whatsapp account with Waboxapp account.

## API

Reference API docs [here](https://www.waboxapp.com/assets/doc/waboxapp-API-v3.pdf)
